Ejemplo de JSON para reportar una incidencia ciudadana
======================================================

Ruta: POST http://localhost:3000/incidencia

JSON que enviaría un ciudadano:
{
  "tipo": "Iluminación pública",
  "descripcion": "La comunidad reporta que una lámpara del parque central no funciona desde hace varios días, afectando la seguridad de los vecinos durante la noche."
}

Respuesta esperada del servidor:
{
  "mensaje": "Incidencia registrada",
  "tipo": "Iluminación pública",
  "descripcion": "La comunidad reporta que una lámpara del parque central no funciona desde hace varios días, afectando la seguridad de los vecinos durante la noche."
}

Otros ejemplos de incidencias que los ciudadanos podrían reportar:

1. Daño en vía pública:
{
  "tipo": "Daño en vía pública",
  "descripcion": "Existe un hueco profundo en la calle 5 con carrera 8 que representa un peligro para vehículos y peatones."
}

2. Basuras acumuladas:
{
  "tipo": "Recolección de basuras",
  "descripcion": "Lleva tres días sin pasar el camión recolector por el barrio El Pozón y los residuos están acumulados en la esquina."
}

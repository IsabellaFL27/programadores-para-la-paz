Prueba de la ruta POST /registro
=================================

Se envió una petición POST a: http://localhost:3000/registro

Con el siguiente JSON en el cuerpo (body) de la petición:
{
  "nombre": "Isabella",
  "mensaje": "Hola comunidad"
}

Respuesta que devuelve el servidor:
{
  "estado": "Datos recibidos",
  "nombre": "Isabella",
  "mensaje": "Hola comunidad"
}

Explicación:
El servidor recibe los datos gracias al middleware express.json(), que permite
leer el cuerpo de la petición en formato JSON. Luego accede a cada campo a
través de req.body.nombre y req.body.mensaje, y los devuelve al cliente
dentro de un nuevo objeto JSON como confirmación de que los datos llegaron
correctamente.
